# Teste-de-Formulario

Formulário de Contato
Este projeto é um exercício acadêmico desenvolvido para praticar conceitos básicos de **HTML** e **CSS**, criando um formulário simples, responsivo e organizado.

Funcionalidades do formulário

* Campo de texto para **nome**
* Campo para **e-mail**
* Campo de **senha**
* Seleção de **gênero**
* Escolha de **turno preferido** (manhã, tarde ou noite)
* Área de texto para **mensagem**
* Botão de envio estilizado

Objetivos do exercício

* Aplicar fundamentos de formulários HTML (`input`, `select`, `textarea`)
* Trabalhar propriedades de CSS para layout, espaçamento e responsividade
* Estilizar interações com foco (`focus`) e estado de botão (`hover`)

Tecnologias utilizadas

 Tecnologia | Utilização                        

 HTML5      | Estrutura do formulário e campos  
 CSS3       | Estilização, espaçamento e layout 

 Como visualizar

1. Baixe o arquivo `index.html`
2. Abra no navegador de sua preferência
